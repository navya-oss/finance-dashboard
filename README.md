# 💰 Finance Dashboard

A modern Finance Dashboard built using React, Tailwind CSS, and Recharts.

## 🚀 Features

- 📊 Dashboard with Balance, Income, Expenses
- 🥧 Pie Chart (Income vs Expense)
- 📈 Line Chart (Spending Trend)
- ➕ Add Transactions (Admin role)
- 🔍 Search & Filter Transactions
- 💡 Insights (Highest Expense, Totals)
  
## 🛠️ Tech Stack
- React
- Tailwind CSS
- Recharts
